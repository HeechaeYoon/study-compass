# 자기주도학습 성향 진단 앱

중학교 2학년 자기주도학습 수업을 위한 정적 웹앱입니다. 학생이 16문항에 답하면 현재 답변 기준으로 가까운 학습 성향, 5개 자기주도학습 축, 성장 포인트, 추천 학습법, AI 챗봇 입력용 프롬프트를 확인할 수 있습니다.

이 앱은 성격검사나 심리검사가 아닙니다. 16문항 기반의 짧은 수업용 자기이해 활동이므로, 검증 척도 수준의 신뢰도와 타당도를 보장하지 않습니다. 결과는 현재 답변 기준의 참고 자료이며, 과목, 단원, 컨디션, 최근 학습 경험에 따라 달라질 수 있습니다.

## 로컬 실행

```bash
npm install
npm run dev
```

개발 서버가 출력하는 로컬 주소로 접속합니다.

## 테스트

```bash
npm test
```

점수 계산, 유형 매칭, AI 프롬프트 생성 로직을 Vitest로 검증합니다.

## 빌드

```bash
npm run build
```

빌드 결과는 `dist/`에 생성됩니다.

## GitHub Pages 배포

`vite.config.ts`에서 `base: "./"`를 사용하므로 저장소 경로에 덜 의존합니다.

1. `npm run build`를 실행합니다.
2. `dist/` 폴더를 GitHub Pages 배포 대상으로 설정합니다.
3. GitHub Actions를 쓴다면 Node 설치, `npm ci`, `npm run build`, Pages 업로드 순서로 구성합니다.

## 개인정보 처리 원칙

- 서버 API, DB, 로그인, 외부 AI 호출을 사용하지 않습니다.
- 학생의 답변, 결과, 메모를 외부로 전송하지 않습니다.
- 저장 기능은 브라우저의 `localStorage`만 사용합니다.
- 공용 태블릿에서는 저장된 결과가 남을 수 있으므로 사용 후 삭제해야 합니다.
- 저장된 결과와 메모는 앱 안에서 삭제할 수 있습니다.

## 주요 기능

- 16문항 기본형 응답
- 5개 자기주도학습 축 점수와 라벨 표시
- 대표 성향 및 보조 성향 처리
- 상세 리포트 생성 및 복사
- AI 챗봇 입력용 프롬프트 생성 및 복사
- 결과 저장, 저장 목록 보기, 개별/전체 삭제
- 결과 요약 카드 이미지 저장 시도
